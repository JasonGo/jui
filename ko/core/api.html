API

JUI 라이브러리는 글로벌 객체인 jui를 제공하고, 이 객체를 사용하여 각각의 컴포넌트들을 생성할 수 있습니다.

Before ready

JUI는 ready 시점 이전에 아래와 같은 메소드와 프로퍼티를 제공합니다.
JUI의 포함된 모든 컴포넌트들은 ready의 콜백 함수가 호출된 이후에 생성이 가능합니다.

Name Type Description
ready method UI 컴포넌트들을 사용하기 위해 준비하는 메소드입니다.
defineUI method UI 컴포넌트를 정의하는 메소드입니다.
define method 상속이 가능한 클래스나 객체를 정의하는 메소드입니다.
include method 'define'이나 'defineUI'로 정의된 클래스나 객체를 가져옵니다.
includeAll method 'define'이나 'defineUI'로 정의된 모든 클래스와 객체를 가져옵니다.
log method 로그를 볼 수 있는 툴을 팝업으로 띄우는 메소드입니다.
logUrl property 로그 툴 페이지의 경로를 설정하는 프로퍼티입니다.

After ready

Name Parameter Return Description
add uiObj none 생성한 컴포넌트 객체를 추가합니다.
get index or uiType object or array 현재 생성된 컴포넌트의 객체를 가져옵니다.
getAll none array 현재 생성된 모든 컴포넌트 객체를 가져옵니다.
size none int 현재 생성된 컴포넌트 객체의 개수를 가져옵니다.
addClass uiCls none 컴포넌트 클래스를 추가합니다.
getClass index or uiType object or array 컴포넌트 클래스를 가져옵니다.
getClassAll none array 모든 컴포넌트 클래스를 가져옵니다.
create uiType, selector, options object ready 시점 이후에 컴포넌트를 동적으로 생성할 수 있습니다.
debugAll callback none 현재 생성된 모든 컴포넌트 객체를 디버깅합니다.
