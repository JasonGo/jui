Tab

Tap, top

첫번째는 일반적으로 사용되는 탭 유형입니다. 사용시 유의해야 할 점은 기본 형태이지만 tab-top 클래스를 명시해줘야 합니다.

  • Home
  • CSS
  • Script
  • Menu 

<ul class="tab tab-top">
	<li class="active">
		<a href="#">Home</a>
	</li>
	<li><a href="#">CSS</a></li>
	<li><a href="#">Script</a></li>
	<li class="menu menu-keep">
		<a href="#">Menu <i class="icon-arrow1 icon-white"></i></a>
	</li>
</ul>

Pill, top

탭과 같이 현재 컨텐츠 영역을 구분할 수 있는 선이 하단에 표시됩니다.

  • CSS
  • Script
  • Menu
